# Java Chat Backend System

A simple **console-based chat system** built in **Java** to practice backend development and core programming concepts.  
This project simulates how a basic chat backend works using **Object-Oriented Programming and Java Collections**.

## Features
- User registration
- Create and join chat rooms
- Send messages in chat rooms
- Store messages per chat room
- View chat history
- Multiple users in the same room

## Tech & Concepts Used
- Java
- Object-Oriented Programming (OOP)
- Abstraction
- Encapsulation
- Polymorphism
- Java Collections
- ArrayList
- HashMap
